# tiptap-utils

共享工具函数，无内部包依赖，是所有其他 tiptap-* 包的基础层。

## 导出

| 模块 | 说明 |
|------|------|
| **editor** | 编辑器 DOM 元素获取、状态辅助 |
| **node** | 节点遍历、位置查找、类型判断 |
| **selection** | 选区工具（`isNodeTypeSelected` 等） |
| **platform** | 平台检测（macOS / Windows / mobile） |
| **style** | CSS class 工具 |
| **upload** | 文件上传辅助 |
| **url** | URL 解析和验证 |
| **date** | 日期格式化 |
| **constants** | 共享常量（`SELECTION_TOOLBAR_KEEP_OPEN_ATTR` 等） |
